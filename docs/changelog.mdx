<Update label="v0.5.0" description="May 2026">
- **Theming Surface**: `DocsVariant` API with built-in presets — shadcn light/dark, brutalist, and default
- **SEO Meta**: Auto-generated `<title>`, description, OpenGraph, and Twitter card tags from frontmatter
- **Sitemap & Canonical URLs**: `generate_sitemap_xml()` plus conditional `<link rel="canonical">` injection
- **Compile-time Blog Engine**: Featured posts, related posts, and a scroll progress bar
- **Mermaid Diagrams**: ` ```mermaid ` fenced blocks render as interactive diagrams
- **CSS-class Syntax Highlighting**: Code blocks use themable CSS classes instead of inline styles
- **H1 Stripping**: Duplicate H1 headings are stripped automatically when a frontmatter title is present
</Update>

<Update label="v0.2.0" description="January 2026">
- **OpenAPI API Reference**: Per-endpoint pages with method badges
- **Search Modal**: Keyboard navigation with `Ctrl+K` shortcut
- **Mobile Drawer**: Responsive sidebar on small screens
- **Sidebar Navigation**: Improved with collapsible groups
- **Syntax Highlighting**: Updated color theme
- **Page Navigation**: Fixed links not updating on route change
</Update>

<Update label="v0.1.0" description="December 2025">
- **MDX Documentation**: Initial release with full MDX support
- **Sidebar Navigation**: Configurable via `_nav.json`
- **Dark Theme**: DaisyUI-powered dark mode
- **Table of Contents**: On-this-page navigation
- **Code Highlighting**: Syntax highlighting for code blocks
</Update>
