## Code Blocks

Code blocks support syntax highlighting:

### Single Language

```rust
fn main() {
    println!("Hello, Dioxus!");
}
```

### With Filename

```rust filename="main.rs"
use dioxus::prelude::*;

fn main() {
    dioxus::launch(App);
}

#[component]
fn App() -> Element {
    rsx! {
        div { "Hello, World!" }
    }
}
```

### Code Groups

<CodeGroup>
```rust Rust
fn greet(name: &str) {
    println!("Hello, {}!", name);
}
```

```typescript TypeScript
function greet(name: string) {
    console.log(`Hello, ${name}!`);
}
```

```python Python
def greet(name: str):
    print(f"Hello, {name}!")
```
</CodeGroup>

## Combining Components

You can combine components for rich documentation experiences:

<Tip>
The following example shows how to combine steps with code blocks.
</Tip>

<Steps>
  <Step title="Create Component">
    Define your Dioxus component:

    ```rust
    #[component]
    fn Button(label: String) -> Element {
        rsx! {
            button { class: "btn btn-primary", "{label}" }
        }
    }
    ```
  </Step>
  <Step title="Use Component">
    Use it in your application:

    ```rust
    rsx! {
        Button { label: "Click me" }
    }
    ```
  </Step>
</Steps>
